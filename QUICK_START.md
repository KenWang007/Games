## 📲 安装为手机应用

### iOS (iPhone/iPad)
1. 用 Safari 浏览器打开网站
2. 点击底部的 **分享** 按钮 (📤)
3. 向下滚动，选择 **"添加到主屏幕"**
4. 点击 **"添加"**
5. 完成！现在可以像普通App一样使用

### Android
1. 用 Chrome 浏览器打开网站
2. 点击右上角的 **菜单** (⋮)
3. 选择 **"添加到主屏幕"** 或 **"安装应用"**
4. 点击 **"添加"** 或 **"安装"**
5. 完成！应用图标会出现在桌面
