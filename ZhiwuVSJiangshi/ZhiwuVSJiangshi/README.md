# 🌻 植物大战僵尸 - 网页版 🧟

适合10岁以下小朋友的卡通塔防游戏！

## 🎮 游戏介绍

在这个可爱的塔防游戏中，你需要种植各种植物来保护你的家园，阻止僵尸的入侵！

### 特色
- 🎨 可爱卡通风格，画面友好
- 👆 简单点击操作，适合儿童
- 🌱 5种植物，3种僵尸
- 📈 10个关卡，难度递进
- 🎵 欢快的游戏体验

## 🚀 如何开始

### 方法1：直接打开
由于使用了ES6模块，需要通过HTTP服务器运行：

```bash
# 使用Python
cd ZhiwuVSJiangshi
python3 -m http.server 8080

# 或使用Node.js
npx serve .
```

然后在浏览器中打开 `http://localhost:8080`

### 方法2：使用VS Code Live Server
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🎯 游戏玩法

1. **收集阳光** - 点击天空掉落的阳光和向日葵产生的阳光
2. **选择植物** - 点击顶部卡片选择要种植的植物
3. **种植植物** - 点击草坪格子放置植物
4. **消灭僵尸** - 植物会自动攻击僵尸
5. **保护家园** - 不要让僵尸到达左侧！

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript

- ES6 模块化

## 📝 开发说明

本项目使用纯Canvas绘制，所有视觉效果通过代码实现，无需外部图片资源。

### 添加新植物
1. 在 `js/config/plants.js` 添加配置
2. 在 `js/entities/plants/` 创建植物类
3. 在 `js/core/Game.js` 注册工厂

### 添加新僵尸
1. 在 `js/config/zombies.js` 添加配置
2. 在 `js/entities/zombies/` 创建僵尸类
3. 在 `js/systems/WaveSystem.js` 注册类型
