# 🎮 游戏中心 - 快速开始指南

## 🚀 三步开始游戏

### 方法一：一键启动（推荐）⭐

#### macOS / Linux 用户：
```bash
cd Games目录
./start.sh
```

#### Windows 用户：
双击 `start.bat` 文件

### 方法二：直接打开
1. 找到 `Games` 文件夹
2. 双击 `index.html` 文件
3. 在浏览器中选择你想玩的游戏

### 方法三：手动启动服务器
```bash
# 进入Games目录
cd Games

# 使用Python启动（推荐）
python3 -m http.server 8000

# 然后在浏览器中访问
# http://localhost:8000
```

## ❓ 常见问题

### Q: 打开游戏后没有反应？
A: 建议使用本地服务器运行，而不是直接打开HTML文件。

### Q: 如何切换游戏？
A: 点击浏览器的后退按钮，或者重新访问游戏中心首页。

### Q: 游戏数据会保存吗？
A: 是的，游戏进度和最高分会自动保存在浏览器中。

### Q: 支持移动设备吗？
A: 游戏中心界面支持移动设备，但具体游戏的移动端支持请查看各游戏说明。
